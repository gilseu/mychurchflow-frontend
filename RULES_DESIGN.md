# 🎨 MyChurchFlow — Design System

## 📌 Fonte de Verdade
O arquivo `designsystem.html` é a referência visual oficial do projeto.

Todas as interfaces devem seguir este padrão visual.

---

## 🎨 Cores

### Tema Dark (padrão)

- Background principal: `#0D0B12`
- Superfície: `#201C2E`
- Borda: `#2E2842`

### Texto

- Texto principal: `#F0ECF8`
- Texto secundário: `#9E96B8`

### Cores principais

- Primária: `#7B5EA7`
- Primária clara: `#9B7CC7`

---

## ✍️ Tipografia

### Texto geral
- Fonte: `Plus Jakarta Sans`
- Uso:
  - textos
  - botões
  - inputs
  - navegação

### Títulos e destaque
- Fonte: `Cormorant Garamond`
- Uso:
  - headings
  - elementos premium

---

## 🧱 Estrutura

### Container padrão

- Centralizado
- Max-width: `420px`
- Padding: `28px`
- Border-radius: `14px`
- Borda: `1px solid var(--border)`
- Fundo: `var(--surface)`
